# HTML面试题
### 如何理解HTML语义化?
   - 让人更容易读懂(增加代码可读性)
   - 让搜索引擎更容易读懂(SEO)

### 默认情况下, 哪些HTML标签是块级元素、哪些是内联元素
1. **块元素(display:block/table;)独占一行**

   - div
   - section
   - ul ol
   - li
   - h1-h6
   - p
   - table
2. **内联元素(display:inline/inline-block;)不会独占一行**

   - span
   - i
   - input
   - button
   - img
